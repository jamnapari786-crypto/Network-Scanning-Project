# Network Scanning & Enumeration Project (CEH)

## Objective
To perform network scanning and enumeration on an authorized target to identify
open ports, services, and potential security exposures.

## Tools Used
- Nmap
- enum4linux
- theHarvester
- Nessus

## Methodology
1. Host discovery using ICMP (ping)
2. Port scanning using Nmap
3. Service and version detection
4. SMB enumeration
5. Email enumeration using OSINT
6. Vulnerability scanning using Nessus

## Disclaimer
This project was performed on authorized targets strictly for educational purposes.
